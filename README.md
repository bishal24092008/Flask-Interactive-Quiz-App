# Flask Interactive Quiz App

A Flask-based interactive MCQ Quiz web application.  
It contains random questions, score tracking, session handling and result analysis.

## Features
- Random Question Order
- Score Calculation
- Session Based Tracking
- Result Category (Try Again / Good / Excellent)
- Clean UI

## Tech Stack
Python Flask  
HTML  
CSS

## How to Run This Project

1️⃣ Install Flask
pip install flask

2️⃣ Run the app
python app.py

3️⃣ Open in browser
http://127.0.0.1:5000/
